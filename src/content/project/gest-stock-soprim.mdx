# 📦 Gest Stock – Soprim

**Gest Stock Soprim** est un logiciel interne développé pour l’entreprise **Soprim**, destiné à optimiser et automatiser la gestion de son stock.  
Le système centralise les informations sur les produits, les mouvements d’entrées/sorties et permet d’avoir une meilleure visibilité sur les approvisionnements.

## ✨ Fonctionnalités clés
- 📊 Suivi des entrées et sorties de stock en temps réel  
- 📦 Gestion des fournisseurs et des clients  
- 📑 Génération de rapports et statistiques  
- 🖥️ Interface bureautique via **PyQt**  

## 🛠️ Technologies utilisées
- **Backend** : Django (Python)  
- **Interface** : PyQt + CSS pour le style  
- **Base de données** : PostgreSQL  

## 🚀 Impact
- 📉 Réduction des pertes liées aux erreurs manuelles  
- ⏱️ Gain de temps grâce à l’automatisation  
- 🔍 Traçabilité complète des produits
